История создания

PHP, созданный в 1994 году Расмусом Лердорфом, первоначально представлял собой простой набор Common Gateway Interface (CGI) бинарных файлов, написанных на языке программирования C.

Краткий обзор языка

PHP - это популярный открытый язык скриптов, общего назначения, который особенно подходит для веб-разработки и может быть встроен в HTML.
